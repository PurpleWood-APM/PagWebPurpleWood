Arquitectura de Conexión

En la imagen se presenta la arquitectura de conexión utilizada en la primera fase del proyecto integrador. Incluye lo siguiente:

• red UNAL: Contiene la red privada de PLCs del laboratorio de automatización.
• red PC 1: Computador personal usado para ejecutar las tareas correspondientes a Studio 5000, Siemens NX, y Robotstudio. Las cuales, entre otros, permiten modificar el estado de los PLC.
• red PC 2: Computador personal que umple las tareas de monitoreo del estado de los PLC por medio de Ingnition.
• red PC 3: Un dispositivo con acceso a internet
• red Cloud: Servicios creados en la nube para monitorear el ambiente de PLCs.

Las conexiones entre las diferentes redes se hacen de la siguiente manera:

• VPN: Para permitir el acceso de PC 1 y PC 2 a la red privada del laboratorio de la Universidad, se realiza una conexión por medio de la VPN Global Protect.
• HTTP: El acceso a los recursos desplegados en la nube se plantea inicialmente mediante el uso de un protocolo HTTP, que permite ingresar desde cualquier dispositivo con conexión a internet que tenga las credenciales de acceso a la cuenta en la nube.

Como se evidencia en el link de acceso a la página web, un valor agregado dado al sitio es su propio dominio PurpleWood.solutions, el cual, además de brindar una URL más amigable y empresarial, permite incluir el modelo de continuous delivery (CD) en el desarrollo de la página. Mediante el modelo CD, cada que se realiza una actualización al repositorio, se actualiza a su vez el despliegue de la página web en el dominio, esto permite realizar cambios continuos y pequeños que se alinean con el modelo SCRUM adoptado para el proyecto, con el cual se realizan múltiples actualizaciones de la página en cada sprint para finalmente llegar al mínimo producto viable (MVP) establecido para el final del sprint.

El despliegue de la página web se hace usando la herramienta netlify, la cual compila el código alojado en el repositorio de github y lo presenta al dominio. Una guía completa de cómo se realizó el despliegue de la página web usando un dominio propio se puede encontrar en:
